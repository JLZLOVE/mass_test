## 修正后的 VO（仅含已确认公开字段）

### 通知列表 VO

| 字段 | 说明 | 来源 |
| :--- | :--- | :--- |
| `noticeNo` | 通知编号 | `notice_info.notice_no` |
| `title` | 标题 | `notice_info.title` |
| `summary` | 内容摘要（运行时截取前150字符） | `notice_info.content` |
| `publishTime` | 发布时间 | `notice_info.publish_time` |
| `topFlag` | 是否置顶 | `notice_info.top_flag` |
| `coverImage` | 题图路径 | `notice_info.cover_image` |
| `hasAttachment` | 是否有公开附件 | 根据 `attachment_min_level` 判断（null或0为公开） |
| `readCount` | 已读人数 | `notice_read_record` 统计 |
| `receiverCount` | 接收总人数 | `notice_info.receiver_count` |
| `readRate` | 阅读率 | `readCount / receiverCount × 100%` |
| `viewCount` | 浏览量 | `notice_info.view_count` |

### 通知详情 VO

| 字段 | 说明 | 来源 |
| :--- | :--- | :--- |
| `noticeNo` | 通知编号 | `notice_info.notice_no` |
| `title` | 标题 | `notice_info.title` |
| `content` | 完整正文 | `notice_info.content` |
| `publishTime` | 发布时间 | `notice_info.publish_time` |
| `topFlag` | 是否置顶 | `notice_info.top_flag` |
| `coverImage` | 题图路径 | `notice_info.cover_image` |
| `attachments` | 附件列表（仅当有公开附件时返回） | 见下方说明 |
| `readCount` | 已读人数 | `notice_read_record` 统计 |
| `receiverCount` | 接收总人数 | `notice_info.receiver_count` |
| `readRate` | 阅读率 | `readCount / receiverCount × 100%` |
| `viewCount` | 浏览量 | `notice_info.view_count` |

**附件返回规则**：`attachment_min_level` 为 null 或 0 时，附件可公开下载；`attachment_min_level` > 0 时，该附件有最低权限要求，门户不展示（返回空数组）。门户只展示公开附件。
