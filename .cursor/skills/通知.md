## 通知模块完整逻辑

> 与 `PROJECT_OVERVIEW.md` 同步（2026-07-18）

### 一、发送通知

#### 入参

| 字段 | 必填 | 说明 |
|---|---|---|
| 标题 | 是 | 通知标题 |
| 内容 | 是 | 富文本内容 |
| 分类ID | 是 | 关联 `notice_category` |
| 接收范围类型 | 是 | 1全体学生 2全体老师 3指定角色 4指定社团 5指定人员 |
| 接收范围值 | 条件必填 | JSON 数组如 `"[1,2,3]"`；type=3/4/5 时 `assertReceiverValuesValid`，非法 **6414**、空 **6415** |
| 重要程度 | 否 | 3低 2中 1高 |
| 紧急程度 | 否 | 2不紧急 1紧急 |
| 是否需要确认 | 是 | true/false |
| 定时发送时间 | 否 | 为空则立即发送 |
| 是否置顶 | 否 | true/false |
| 置顶结束时间 | 否 | 置顶时必填 |
| 是否长期可见 | 是 | true=新成员也能看 |
| 附件 / 附件权限 | 否 | `attachment_min_level` |

#### 权限校验

| 发送方 | 默认范围 | 跨范围 |
|---|---|---|
| 管理员 | 全校 | 不需审批 |
| 校书记 | 全校 | 跨学院需审批（完整流待做，现多返回 6404） |
| 指导老师 | 所指导社团 | 跨社团/学院需审批 |
| 社长 | 本社团 | 不需审批 |
| 部长 | 本部门 | 发全社团需社长授权 |
