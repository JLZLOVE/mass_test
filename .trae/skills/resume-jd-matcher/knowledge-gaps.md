## 累计学习路线图（按优先级排序）

### 🔴 紧急（面试前1-2天集中补课）
1. JVM 内存模型（堆/栈/方法区/程序计数器）
2. GC 算法（标记清除/复制/标记整理/分代收集/CMS/G1）
3. 类加载机制（双亲委派模型）
4. 多线程基础（线程生命周期、synchronized、volatile）
5. 线程池（ThreadPoolExecutor 七大参数）

### 🟡 高优（1周内）
6. 机器学习/深度学习基本概念
7. IPC 进程间通信（管道/共享内存/消息队列/信号量）
8. JVM 调优参数（-Xms/-Xmx/-Xss）

### 🟢 中优（有空就学）
9. Transformer / GPT 架构基本概念
10. Selenium 前端自动化测试基础
11. PostgreSQL 与 MySQL 的区别
12. MongoDB 文档型数据库基本概念
13. 操作系统深入（进程调度、内存管理、文件系统）

### ⚪ 低优（不急）
14. C/C++ 基础语法
15. PyTorch 入门
16. LangChain / Agent 框架
17. 多模态模型概念
18. Cassandra 分布式数据库
